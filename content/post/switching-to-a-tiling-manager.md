Having been a long time Linux user for my daily driver, I finally decided to make a switch... Which operating system have I chosen to use instead? Well... Linux of course.

I have not  ditched the penguin for another operating system. But what exactly did I switch then? Well, I am no longer using a traditional Desktop Environment (DE). I switched over to a Tiling Window Manager (WM). Now, which WM did I go with? Well, that was the hardest part... there are so many great ones out there. After taking some time, about 2 minutes, I decided to use the qTile Window Manager.

Being primarily a Python developer, I figured, what better window manager to use than qTile, as it's not only written in Python, but it is also configured with Python. We will see how it goes from here!
